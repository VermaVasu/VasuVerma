Projects

Debug HSSI between DLP controller and DMD

Worked on debugging of Texas Instruments Proprietary High speed HSSI signals(3.2GHz) flowing between the DLP(Digital Light Processing) Controller and the DMD(Digital Micromirror Device). Characterization of the signal by providing an eye diagram by solely running python scripts on the Display Controller Software and Embedded C functionalities on the circuit board. Eliminated the use of oscilloscopes to plot an eye diagram while simultaneously providing a solution to improve the signal for the consumers.

ARDUINO BASED LI-FI PROTOTYPE

LiFi is a fundamental 5G technology. With claims to be 100 times faster and theoretical speed limit of 100Gbps, this technology will clearly unlock the IoT and drive Industry 4.0 applications. Did a college funded project on ways to improve the speed of data transmission using modulation schemes on MATLAB and indigenously developed Li-Fi based transceivers.

Exploring biomarkers for early detection of COVID

Data analysis on Patient’s Sp02 measurements found using Pulse Oximetry. Studied it’s direct relation for detecting early onset of COVID Pneumonia to assist in time quarantining and hospitalization of at risk individuals.

BOLT IOT AND ML PROJECTS

Worked on Bolt IoT platform based several projects implemented by C++, Python and APIs - Polyfit and Z-Score. Helped me gain knowledge about integration of several devices using IoT and ML. Did various projects regarding implementation of the same. A list of projects includes Home Automation, Voice Controlled Lighting, Environmental Monitoring, Smart Farming, etc.

Optimization of 2 stage operational amplifier

Designed a Two stage single ended output Differential Amplifier (differential + Gain stage) with power dissipation less than 200nW. The main features achieved through the design were Slew Rate greater than 10V/uS and Gain Bandwidth = 5MHz using LTSpice tools
